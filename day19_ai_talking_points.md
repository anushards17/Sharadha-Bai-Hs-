# AI Talking Points Card

## Groq
Groq is the AI service used in this project to generate fast responses from prompts.  
It helps the system understand policy-related questions and return intelligent answers quickly.

## RAG (Retrieval-Augmented Generation)
RAG combines stored document knowledge with AI-generated responses.  
It first searches related policy data from ChromaDB and then sends that context to the AI model for better answers.

## ChromaDB
ChromaDB is used as the vector database in this project.  
It stores document embeddings and helps retrieve the most relevant policy content during searches.

## Prompt Engineering
Prompt engineering means writing clear and structured instructions for the AI model.  
Better prompts improve response accuracy, formatting, and overall output quality.
